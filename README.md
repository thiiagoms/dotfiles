### Dependencies :hammer:
- Bash/Zsh
### Use :package:

01 - Clone:
```bash
$ git clone https://github.com/thiiagoms/dotfiles
```

02 - Change to `dotfiles` directory:
```bash
$ cd dotfiles 
dotfiles $
```

03 - Execute `arch.sh` for Arch Linux / Manjaro distro
```bash
$ ./arch.sh
```
